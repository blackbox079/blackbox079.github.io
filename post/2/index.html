因为希望提高阅读体验而尝试自制epub文档，后续会继续完善。主要使用 VSCode 和 Sigil 。

准备工作

使用 VSCode 整理文档。

  • 清除段前空格 : 搜索 　　 替换为空
  • 清除多余换行 : 搜索 \n\n 替换为 \n
  • 校正错漏 ：错别字，被和谐的字符 * 或 □ ，混用的字符比如逗号句号冒号双引号。
  • 标准化卷、章、节：统一文档内关键字，方便批量添加标记。

添加标记

  • 添加 p 标签 搜索 (.*) 替换为 <p>$1</p>
  • 处理卷 搜索 <p>(第.*卷.*|卷.*)</p> 替换为 <hr class="sigil_split_marker"/>\n<h1>$1</h1>
  • 处理章 搜索 <p>(第.*章.*|番外.*|序章)</p> 替换为 <hr class="sigil_split_marker"/>\n<h2>$1</h2>

其中， <hr class="sigil_split_marker"/> 是 Sigil 的切分标记。

处理内容提要

方法1：

  • 搜索 <h2>(第.*章.*|番外.*) (.*)</h2> 替换为 <h2>$1</h2>\n<div class="summary">\n<p>$2</p>\n</div>

方法2：

  • 搜索 (第\d\d章.*) 替换为 $1\n==== ，然后将提要补充到等号中间，即 ==提要==
  • 搜索 <p>==(.*)==</p> 替换为 <div class="summary"><p>$1</p></div>

制作epub

Sigil是开源的电子书制作软件，软件安装包可在 Github Releases 下载。

打开 Sigil ，默认页面的一本全新的电子书，将在 VSCode 处理好的文档复制粘贴到 Text\Section0001.xhtml 的 <body></body> 标签里。使用快捷键 F6 在标记处切分，即可把每一章划分成一个xhtml文件。检查一下是否有章节划分错误，然后对这部分文件重命名。

  • Cover 封面
  • Introdution 扉页，简介，详情
  • Nav 目录
  • Volume 卷
  • Chapter 章
  • Gaiden 番外
  • Version 版本及更新信息

通常切分完成之后 Section0001.xhtml 内容是空的，可以直接重命名为 Introduction，放入书籍相关介绍。Cover.xhtml 会在添加封面之后自动生成。Version.xhtml 可在Text目录下右键选择 添加空白HTML文件 ，后续补充版本更新日志。

添加封面和元数据

如果有封面的话，可通过 工具-添加封面 进行添加。快捷键 F8 编辑元数据，通常包括：

  • dc:title 书名
  • dc:creator 作者
  • belongs-to-collection 作品集
  • dcterms:created 发表时间
  • dc:language 语言，通常是 zh-CN
  • dc:source 来源
  • dc:description 描述
  • meta 自动生成创建时间

完善title和添加样式

可以删除 Styles 目录下的 sgc-nav.css 文件，然后右键添加现有文件，选择常用的css文件。（css文件重复使用） 搜索 (?s)<title></title>(.*)<h2>(.*)</h2> 替换为 <title>\2</title>\n<link href="../Styles/main.css" rel="stylesheet" type="text/css"/>\1<h2>\2</h2>

处理作话

方法1： 搜索 (?s)<p>作者有话要说：</p>(.*)</body> 替换为 <div class="authorSayBox">\1</div>\n</body>

方法2： 搜索 (?s)<p>===</p>\n(<p>.*</p>)\n<p>===</p> 替换为 <div class="authorSayBox">\n\1\n</div>
